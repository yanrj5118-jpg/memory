# 🛍️ 올리브영 수익화 쇼츠 (Commerce Shorts Architect)

이 워크플로우는 올리브영 상품 링크를 분석하여 시청자의 구매 욕구를 자극하고, 실제 수익으로 연결되는 쇼츠 대본을 기획하는 데 사용됩니다.

## 📋 핵심 실행 단계

### 1. 상품 포인트 추출 (Link Analysis)
- 제공된 올리브영 URL을 기반으로 상품의 **핵심 성분, 특징, 실제 단점(솔직함)**을 분석합니다.
- 현재 진행 중인 **세일 정보**나 **증정품 혜택**을 파악하여 대본의 긴박함을 더합니다.

### 2. 타겟 페르소나 매칭
- 이 제품이 '누구'에게 가장 필요한지 정의합니다. (예: 아침마다 붓는 사람, 여드름 흉터가 고민인 사회초년생 등)
- 해당 타겟이 2026년에 가장 많이 사용하는 언어와 유머 코드를 적용합니다.

### 3. 고전환 수익화 대본 구조
- **0-3s (Pain Point Hook):** 시청자가 겪고 있는 고민을 즉각적으로 건드립니다. (예: "아직도 이거 없어서 화장 뜨세요?")
- **4-10s (Evidence & Solution):** 제품의 실제 사용 장면(또는 비주얼 프롬프트)과 함께 확실한 효과를 보여줍니다.
- **11-16s (CTA & Urgency):** "지금 올영 세일 기간에만 이 가격"이라는 점을 강조하며 고정 댓글 클릭을 유도합니다.

### 4. 수익화 최적화 가이드
- **링크 배치:** 알고리즘 점수 하락을 방지하기 위해 댓글창 활용 및 고정 댓글 전략을 제안합니다.
- **공정위 문구:** 2026년 강화된 표시광고법에 따른 광고 표기 문구를 자연스럽게 삽입합니다.

## ⚠️ 커머스 핵심 원칙
- **솔직함(Authenticity):** 너무 완벽한 광고보다, 약간의 아쉬운 점을 섞은 '진짜 리뷰' 스타일이 2026년에는 훨씬 잘 팔립니다.
- **데이터 기반:** 리서치를 통해 얻은 실제 평점과 베스트 리뷰 내용을 대본에 녹여 신뢰도를 높입니다.
