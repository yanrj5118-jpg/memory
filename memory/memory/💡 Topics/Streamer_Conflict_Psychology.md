# 스트리머 간의 경쟁 상황 분석
## 📌 한 줄 통찰
> 극단적인 경쟁 환경에서 개인들은 감정을 배제하고 생존을 위해 상대를 목표로 삼으며, 이는 심리적 압박과 전략적 플레이를 통해 공격성으로 표출됨을 보여준다.
## 📖 구조화된 지식
- **갈등의 본질:** 참가자들은 단순히 자신만을 노리는 것이 아니라 '우승을 위협하는 상대'를 목표로 총을 쏘는 등, 상호적인 공격 패턴을 보였다.
- **감정 배제와 행동:** 개인들은 관계에서 감정을 제거하고 목표 달성에 집중하며, 이는 트릭샷과 같은 예측 불가능한 행동으로 나타난다.
- **집단 심리:** 경쟁 상황은 비호감 순위가 드러나게 하며, 결국 일부는 다른 이들을 의심하고 반격하는 형태로 변화한다.
